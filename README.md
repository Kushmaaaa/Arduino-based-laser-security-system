
# Laser-Based Security System using Arduino Uno

A simple Arduino-based security system that uses a laser module and an LDR sensor to detect intrusions. When the laser beam is interrupted, the Arduino activates a buzzer to alert the user. The project also includes an ON/OFF switch to enable or disable the system when needed.

## Components

* Arduino Uno
* Laser Module
* LDR Module
* Buzzer
* ON/OFF Switch
* Breadboard
* Jumper Wires

## Features

* Laser beam intrusion detection
* Buzzer alert
* ON/OFF control
* Easy to build and cost-effective
